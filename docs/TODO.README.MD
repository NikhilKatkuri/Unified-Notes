# Authentication Endpoints

This document outlines all authentication-related endpoints.

## Endpoints

| Endpoint Path  | HTTP Method |
| -------------- | ----------- |
| /signup        | POST        |
| /signin        | POST        |
| /find          | GET         |
| /account       | DELETE      |
| /user          | PUT         |
| /logout        | POST        |
| /refresh       | POST        |
| /reset-request | POST        |
| /reset-confirm | POST        |
| /verify        | GET         |
| /me            | GET         |
| /users         | GET         |
